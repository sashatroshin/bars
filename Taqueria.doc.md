# Taqueria/Bar/taqueria-bar-mikroinventa-*.html — Микроинвентаризация Такерии

**Фича:** HTML-формы для ежедневной/сменной микроинвентаризации в баре Taqueria. По файлу на день недели:
- cht (четверг), pn (понедельник), pt (пятница), sb (суббота), sr (среда), vs (вторник), vt (воскресенье)

**Поток:** Открыть форму нужного дня → заполнить остатки по позициям → сохранить/localStorage.

**Что важно:** Все формы самодостаточны (нет внешних CSS/JS). vs (вторник) — самая большая форма (22КБ, много позиций). Остальные — унифицированные по 13КБ. Данные хранятся в localStorage браузера.
