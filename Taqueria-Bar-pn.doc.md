# Taqueria/Bar/taqueria-bar-mikroinventa-pn.html — Микроинвентаризация Такерии (понедельник)

**Фича:** Форма инвентаризации на понедельник для бара Taqueria.

**Поток:** Открыть → заполнить остатки по каждой позиции → автосохранение в localStorage.

**Что важно:** Увеличенная форма (21КБ) — пн требует больше позиций, чем другие дни (кроме вт). Содержит те же поля, что и остальные дни, но с другим набором.
