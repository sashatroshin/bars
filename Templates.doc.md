# Templates/subscription-form-template.html — Шаблон формы подписки

**Фича:** Компактный шаблон формы email-подписки с карточкой и кнопкой. Uiverse-стиль.

**Поток:** Встроить в HTML-страницу → пользователь вводит email → отправка.

**Что важно:** Английский интерфейс. Чистый CSS без внешних зависимостей. Минималистичный дизайн (240px карточка). Пример для копирования.
